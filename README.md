# Ecommerce Sales Dashboard

## Overview
Developed an Ecommerce Sales Dashboard using Google Sheets for visualization, SQL views for data aggregation, and PowerShell automation for semi-automated data integration.

## Tools Used
- Google Sheets
- SQL
- PowerShell

## Key Metrics
- Total Revenue
- Customer Ratings
- Cancellation Rate
- Monthly Sales Trend
